# electron-angularjs-tabs
Tab based application with Electron, AngularJS and Bootstrap.
[Clube dos Geeks](//clubedosgeeks.com.br)

## Use
### 1 - Clone
```shell
git clone https://github.com/ClubeDosGeeksCoding/electron-angularjs-tabs.git
```

### 2 - Install
```shell
npm install
```

### 3 - Start
```shell
npm start
```
